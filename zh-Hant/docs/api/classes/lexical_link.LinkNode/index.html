▸ createDOM(config): LinkHTMLElementType

updateDOM​

▸ updateDOM(prevNode, anchor, config): boolean

當節點發生變更並且需要更新 DOM 以使其與更新期間可能發生的變化對齊時，將調用此函式。

如果返回 "true"，則會導致 lexical 卸載並重新創建 DOM 節點（通過調用 createDOM）。例如，當元素標籤更改時，您需要這樣做。
